# 🎶 MusicBoxd  

Uma aplicação em **React** para explorar músicas, artistas e álbuns em uma interface moderna e intuitiva.  
O projeto foi desenvolvido como forma de praticar conceitos de **frontend**, integração de componentes e usabilidade.  

---

## 🚀 Tecnologias  

- ⚛️ [React](https://react.dev/)   
- 🔄 [Vite](https://vitejs.dev/)  
- 🧩 Componentes reutilizáveis  

---

## 📂 Estrutura do Projeto  

MusicBoxd/
│── src/

│ ├── pages/ # Páginas principais
│ ├── assets/ # Ícones, imagens, fontes

│
│── public/ # Arquivos estáticos

│── README.md

## ⚡ Como rodar o projeto  

Clone o repositório e entre na pasta:  

```bash
git clone https://github.com/leonard0antonio/MusicBoxd.git

cd MusicBoxd

Instale as dependências:

npm install

Inicie o servidor local:

npm run dev

Acesse no navegador:
👉 http://localhost:5173
